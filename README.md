# 非暴力沟通 AI 助手

一个基于通义千问的智能聊天助手，专门帮助用户学习《非暴力沟通》中的概念和技巧。

## 功能特点

- 🤖 智能对话：基于通义千问大模型
- 📚 知识库集成：结合《非暴力沟通》知识库
- 💬 流式输出：实时显示 AI 回复
- 🎨 现代界面：深色主题，响应式设计
- 📱 移动端适配：支持手机和平板

## 技术栈

- 后端：Python + Flask
- 前端：HTML + CSS + JavaScript
- AI：通义千问 API
- 部署：Vercel

## 本地运行

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行服务器：
```bash
python app.py
```

3. 访问：http://localhost:8080

## 部署

本项目已配置支持 Vercel 一键部署。
